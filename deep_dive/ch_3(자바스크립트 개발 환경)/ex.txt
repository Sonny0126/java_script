렌더링된 뷰를 확인할 때, Elements 확인
Console 패널은 에러가 났을 때 가장 먼저 찾아보기
Sources는 웹페이지의 자바스크립트 코드를 디버깅
Network는 웹페이지 네트워크 요청정보, 성능 확인
Application은 웹 스토리지, 세션, 쿠키 확인 및 관리

npm(node package manager)은 Node.js에 사용할 수 있는 모듈들을 모아둔 저장소

버전 확인 
node -v
npm -v  

확장 플러그인
code runner 설치
Ctrl + Alt + N 을 사용해 js 코드 실행 가능

Live Server에서 가상서버 확인 가능
